# Personal Portfolio Webpage

A sleek, modern, and fully responsive personal portfolio webpage developed using semantic HTML5 and pure CSS3 properties. The site is tailored to showcase academic qualifications, technical competencies, and software engineering projects.

## 🚀 Features
* **Semantic HTML5:** Structured using clean accessibility standards (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`).
* **Modern CSS Layouts:** Designed using **CSS Flexbox** for navigation and hero layouts, and **CSS Grid** for the responsive project displays.
* **Tabular Highlights:** Clean, custom-styled data tables outlining career experience and educational qualifications.
* **Circular Profile Accent:** Styled user profile frame with a distinct multi-layered purple aesthetic border ring.
* **Fully Responsive:** Adapts seamlessly across mobile, tablet, and desktop viewports.

## 🛠️ Built With
* **HTML5** - Page structure and document flow.
* **CSS3** - Custom variables, responsive typography, Flexbox, and Grid layouts.
